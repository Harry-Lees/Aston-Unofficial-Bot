FAQ

Can I add this bot to my server?

If you're in any way affiliated with Aston Univeristy, you're eligible to have this bot on your server. Please get into contact with us with the form below.

How does the verification work?

Our verification system works by requesting users sign in with their email and then assigning them a role once they've input their email address and clicked on an email link.

How can we trust you?

This project is entirely open source, you can access the code through the Github link in the navigation bar, meaning you have complete transparency about our system.

Are you affiliated with Aston Univeristy?

This project is maintained in it's entirety by students at Aston University. We are in no way officially affiliated with the Univeristy.

Anything else?

If you have any other queries about this bot, please contact Going_Quantum #6266 on Discord.
If you would like to contribute to this project, please open an issue on Github for discussion.
